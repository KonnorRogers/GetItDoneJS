constant pipe

pipe() is a utility function which will add multiple functions onto it. The functions passed in should take in an {Object} parameter. This function is similar to lodash's flow function. It is taken from Eric Elliot.

See:
  • pipe function
  • ItemMixin for further usage

View Source utils/pipe.js, line 20

Example
const createDrone = ({capacity = '3000mAh'}) => pipe(
    withFlying,
    withBattery({capacity}),
    withConstructor(createDrone)
  )({});
