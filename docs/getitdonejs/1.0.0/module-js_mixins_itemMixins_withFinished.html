js/mixins/itemMixins/withFinished

This provides utility functions for finished property.

Parameters:
Name Type Attributes Default Description
props Object <optional>
 {}

The properties {Object} to pull from
finished boolean <optional>
 false

Boolean value of if the item is finished
obj Object

The object to append the methods onto

View Source mixins/itemMixins/withFinished.js, line 1

Returns an {Object} with methods appended.

Object
Example
import {withFinished} from '<path>/<to>/withFinished.js';
let item = withFinished()({});
item.isFinished();
// => false

item.isIncomplete();
// => true

item.finish();
item.isFinished();
// => true

item = withFinished({finished: false})({});
item.isFinished();
// => false
